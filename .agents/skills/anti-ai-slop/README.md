# Anti-AI-Slop

Systematic antidote to generic "AI slop" aesthetics in UI, UX, and copy.

## Features
- Checklists for identifying 2025–2026 AI-slop patterns
- Specific "what to do instead" guidance for visual design and interaction
- Principles for human-centric copy and text
- Audit workflow for quality enforcement

## Usage
Activate when auditing or creating UI, UX, or copy to ensure distinctive, high-quality results.
