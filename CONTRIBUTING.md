# Contributing

Thank you for your interest in contributing! Here are some guidelines.

## How to contribute

- Fork the repository or use the template.
- Create a feature branch (e.g., `feature/awesome-thing`).
- Write descriptive commit messages.
- Ensure any new script or feature is documented in `AGENTS.md`
- Run the label initialization script if you add new labels.  
- Submit a pull request. Make sure to fill out the PR template (if present).
